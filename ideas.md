# BIST Doktoru - Tasarım Fikirleri

## Seçilen Yaklaşım: "Bloomberg Terminal Meets Modern Fintech"

### Design Movement
**Dark-Mode Finans Terminali + Modern Minimal Fintech**

### Core Principles
1. Veri yoğunluğu ve okunabilirlik dengesi — her piksel bilgi taşımalı
2. Güven ve otorite hissi — kurumsal ama soğuk değil
3. Gerçek zamanlı dinamizm — canlı, nefes alan arayüz
4. Türk yatırımcısına özgü kimlik — BIST Doktoru markası güçlü

### Color Philosophy
- **Ana Arka Plan:** Derin lacivert-siyah (#0A0E1A) — güven, derinlik
- **Birincil Vurgu:** Elektrik mavisi (#00D4FF) — teknoloji, canlılık
- **Yükseliş Rengi:** Zümrüt yeşili (#00C896) — pozitif piyasa
- **Düşüş Rengi:** Kırmızı (#FF4757) — negatif piyasa
- **Kart Arka Planı:** Koyu gri (#111827) — katmanlı derinlik
- **Metin:** Beyaz ve açık gri tonları

### Layout Paradigm
- Sol kenar çubuğu navigasyon (sabit)
- Sağda geniş içerik alanı
- Üstte canlı ticker şeridi
- Dashboard grid sistemi — farklı boyutlarda widget kartlar

### Signature Elements
1. Canlı fiyat ticker şeridi (üst bar)
2. TradingView grafik widget'ları (tam entegrasyon)
3. Neon glow efektli kart kenarları (yükseliş/düşüş durumuna göre)

### Interaction Philosophy
- Hover'da kart hafifçe yükselir (translateY -2px)
- Fiyat değişimlerinde renk animasyonu (flash effect)
- Smooth scroll ve sayfa geçişleri

### Animation
- Ticker şeridi sürekli kayar
- Sayılar değişirken kısa flash animasyonu
- Sayfa yüklenirken fade-in + slide-up

### Typography System
- **Başlıklar:** Space Grotesk (bold, modern fintech)
- **Veri/Sayılar:** JetBrains Mono (monospace, terminal hissi)
- **Gövde:** Inter (okunabilirlik)
